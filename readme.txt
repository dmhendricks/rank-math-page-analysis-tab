An add-on for the Rank Math WordPress plugin that moves On-Page Analysis from the General tab to its own.

== Description ==

This plugin is an simply add-on for the [Rank Math](https://wordpress.org/plugins/seo-by-rank-math/) plugin. It moves On-Page Analysis from the General tab to its own.

GitHub repo: [https://github.com/dmhendricks/page-analysis-tab-rank-math](https://github.com/dmhendricks/page-analysis-tab-rank-math)

**Requirements**

* WordPress 4.9 or higher
* PHP 5.6 or higher
* [Rank Math](https://wordpress.org/plugins/seo-by-rank-math/) WordPress plugin installed and activated

**GDPR/CCPA Safe**

This plugin _does not_ use or track _any user data_, either from visitors or users of WP Admin.

== Installation ==
1. Simply download, install and activate.

Once activated, it automatically sorts sites and adds a filter search input. No configuration needed.
